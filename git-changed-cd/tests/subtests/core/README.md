# Core Functionality Tests

Tests for the original `git-changed-cd` functionality:

- **not_in_git_repo**: Behavior outside Git repositories
- **clean_repo**: Handling repositories with no changes  
- **unstaged_file_navigation**: Navigation to directories with unstaged files
- **staged_file_parent_dirs**: Detection of staged files and parent directories
- **untracked_file_navigation**: Navigation to directories with untracked files
- **debug_mode**: Debug output verification
