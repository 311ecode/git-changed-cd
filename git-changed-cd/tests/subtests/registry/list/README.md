# Registry List Tests

Tests for the repository listing functionality:

- **list_repos_empty**: Behavior when no repositories are registered
- **list_repos_single**: Display of single registered repository
- **list_repos_multiple**: Display of multiple registered repositories with proper numbering
- **list_repos_help**: Help message display for list command
- **list_repos_invalid_parameter**: Invalid parameter handling
- **list_repos_alias**: Verification that `gcd-list` alias works correctly

The list functionality provides users with a way to see all their registered repositories in a numbered format, making it easy to understand what repositories are available for multi-repo navigation.
